How RED Works

RED (Rating of Elite Dominance) measures how much each NBA coach is outperforming or underperforming what their team was expected to do.

The Concept

A coach with RED = +3.0 has generated 3 more wins than expected given their roster quality and preseason expectations. Negative RED indicates underperformance relative to expectations.

Data Inputs

  • Vegas Win Totals: Preseason consensus over/under projections
  • Player Quality (BPM): Basketball Plus-Minus estimates team strength
  • Game Results: Live data from ESPN API (2025-26 season)

Calculation Layers

Layer 1 - Baseline Wins:
Baseline = (0.7 × Vegas Total) + (0.3 × BPM-Derived Wins) BPM Wins = 41 + (Team BPM × 5)
Layer 2 - Game Expected Win Probability:
Strength Diff = Team BPM - Opponent BPM if Home: Strength Diff += 0.5 P(Win) = 1 / (1 + e^(-0.35 × Strength Diff))
Layer 3 - RED Score:
Game Delta = Actual Result (1 or 0) - P(Win) if Margin ≤ 6 pts: Game Delta *= 1.10 (close game bonus) RED = Σ Game Deltas (all games)

Interpretation

Positive RED: Coach exceeded expectations. Team is winning more games than talent level predicted.

Negative RED: Coach underperformed expectations. Team is winning fewer games than roster quality suggested.

Important Notes

  • RED is descriptive, not predictive of future performance
  • Accounts for home court advantage and roster quality differential
  • Close games (≤6 points) weighted higher to reward clutch coaching
  • Only includes completed games from the 2025-26 season
